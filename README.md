# Ray Tracing

## Prerequisite

- [CMake](https://cmake.org/download/)

- GCC / CLANG complier

## How to build project using script

- Window: Build.bat

- Mac/Linux: Build.sh

## How to run project using script

- Window: Run.bat

- Mac/Linux: Run.sh

## How to manual build project and run

- mkdir build
- cd build
- cmake ..
